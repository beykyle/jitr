# Documentation site for `jitr`

`jitr` is a calculable R-matrix toolkit for nuclear-reaction modeling,
uncertainty propagation, and optical-potential workflows.

## Documentation

If you are new to the project, start with
[Getting started](getting-started.md), then browse the curated
[example notebooks](examples/index.md), and use the
[API reference](api/index.md) when you want implementation details.
